\# Retail Ops Dashboard



A single-page application that provides real-time visibility into inventory risks. It consumes JSON data from the `inventory-api` microservice and renders actionable alerts for operations teams.



\## Tech Stack

\- Vanilla JavaScript (ES6+), HTML5, CSS3

\- Fetch API (Async/Await)



\## Usage

1\. Ensure the backend service is running:

&nbsp;  python -m uvicorn app.main:app --reload



2\. Open `index.html` in any modern web browser.



3\. Click \*\*Refresh Data\*\* to pull live records.
